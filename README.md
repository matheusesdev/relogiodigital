# Relógio Digital

Este é um projeto simples de um relógio digital criado com HTML, CSS e JavaScript. O projeto exibe a hora atual e atualiza em tempo real.


## Funcionalidades

- Exibe a hora atual com precisão.
- Atualiza automaticamente a cada segundo.
- Design minimalista e atraente.

## Tecnologias Utilizadas

- HTML
- CSS
- JavaScript
